- **Claude Haiku 3 廃止予定**（`claude-3-haiku-20240307`）: **2026/04/19 にリタイア**。Claude Haiku 4.5 への移行が必要。

- **オーストラリア政府とのMOU締結**（3/31）: Anthropic がオーストラリア政府とAI安全性・研究に関する覚書を締結。

## 【参考】直近の主要リリース（過去1〜2ヶ月）

### 🧠 新モデル

- **Claude Opus 4.6** (`claude-opus-4-6-20250205`、2/5リリース): 最上位モデル。アダプティブ思考推奨、1Mコンテキスト対応、拡張思考サポート。
- **Claude Sonnet 4.6** (`claude-sonnet-4-6-20250217`、2/17リリース): バランスモデル更新。エージェント検索性能向上、トークン消費削減。

### 📡 API変更（3月）

- **3/18**: Models API に `max_input_tokens`・`max_tokens`・`capabilities` フィールド追加。`GET /v1/models` で取得可能。
- **3/16**: 拡張思考の `display: "omitted"` パラメーター追加。ストリーミング高速化。
- **3/13**: Opus 4.6 / Sonnet 4.6 の1Mコンテキストが **GA（正式提供）**。ベータヘッダー不要、標準料金。画像/PDFの上限が100→**600枚**に拡大。
- **2/19**: 自動キャッシュ機能追加（`cache_control` 1フィールドで管理）。

### 🛠️ Claude Code

- Homebrew インストール更新通知の修正、大ファイルの Write ツール diff 計算高速化、フルスクリーンスクロール修正。
- WebFetch が `Claude-User` として識別（robots.txt 許可リスト対応）、ピーク時メモリ削減。
- `/powerup` コマンド追加（アニメーションデモ付きインタラクティブレッスン）。
- **Claude Code Analytics API** 開始: 組織ごとの日次利用メトリクスをプログラム取得可能。

### 🖥️ プロダクト・その他

- **ツール群がGA**: Web Search, Web Fetch, Code Execution, Tool Search, Memory ツールが正式提供（ベータヘッダー不要）。
- **コード実行が無料化**: Web Search / Web Fetch と組み合わせた場合のサンドボックスコード実行が無料。
- **データレジデンシー**: `inference_geo` パラメーターで米国内推論が選択可能（1.1x料金、2/1以降モデル対象）。
