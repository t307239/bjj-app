## 【参考】直近の主要リリース（過去1〜2ヶ月）

### 🧠 新モデル

- **Claude Opus 4.6** (`claude-opus-4-6-20250205`、2/5リリース): 最上位モデル。アダプティブ思考推奨、1Mコンテキスト対応、拡張思考サポート。

- **Claude Code Analytics API** 開始: 組織ごとの日次利用メトリクスをプログラム取得可能。
